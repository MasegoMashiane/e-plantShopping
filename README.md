# e-plantShopping
e-plant is a React-based e-commerce application for buying houseplants.
It includes a product listing page with multiple plan categories, a shopping cart managed with Redux, and a landing page for Paradise Nursery.

##Features
-landing page with welcome message and "get started" button
-Product listing page with plants grouped by categories
-Add-to-cart funcitonalitywith quantity updates.
-Shopping cart page with total amounts, delete, and checkout buttons 
-Redux State management for the cart
-Responsive navbar with links to home, Plants, and cart
